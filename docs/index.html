Deific

Python syntax. C++ speed. No runtime.

Overview

Deific is a transpiler: .df source files are compiled to C++17 and then to a native binary via g++ -O2. It is designed for competitive programming where Python ergonomics are desirable but execution speed is critical.

The pipeline is: .df → Lexer → Parser → AST → Emitter → .cpp → g++ → binary.

The emitted .cpp is self-contained and can be submitted directly to online judges.

Installation

Build from source using the GNU Rust toolchain (MSVC will not work — the compiler invokes g++ at runtime):

cargo +stable-x86_64-pc-windows-gnu build --release

The binary is placed at target/release/deific.exe.

Windows: g++ must be on your PATH. The expected toolchain is MSYS2 ucrt64 (C:\msys64\ucrt64\bin).

CLI Usage

CommandDescription
deific run file.dfCompile and immediately run
deific emit file.dfPrint generated C++ to stdout
deific build file.df -o outBuild to a named binary
deific --versionPrint version

To feed stdin to a compiled program on Windows, use a file redirect to avoid encoding issues:

cmd /c ".\out.exe < input.txt"

Variables

The first assignment in a scope declares the variable. Subsequent assignments reassign it.

x = 10          # declares x (auto)
x = 20          # reassigns x

y: int = 5      # explicit type annotation

Empty lists require a type annotation so the C++ type can be inferred:

nums: list[int] = []

Tuple Unpacking

a, b = 1, 2
a, b = b, a     # swap

Control Flow

if / elif / else

if x > 0:
    print("positive")
elif x == 0:
    print("zero")
else:
    print("negative")

while

while n > 0:
    n -= 1

for

for i in range(10):
    print(i)

for i in range(1, 10, 2):    # start, stop, step
    print(i)

for i, v in enumerate(arr):
    print(i, v)

for a, b in zip(xs, ys):
    print(a, b)

for x in my_list:
    print(x)

break, continue, and pass work as in Python.

Functions

func add(a: int, b: int) -> int:
    return a + b

func greet(name: str) -> void:
    print("Hello", name)

Entry Point

The main function with no parameters and no return type is the program entry point:

func main():
    print("Hello World!")

Reference Parameters

Prefix a parameter with ref to pass by reference (in-place mutation):

func double_all(ref nums: list[int]) -> void:
    for i in range(len(nums)):
        nums[i] *= 2

Inline Functions

inline func square(x: int) -> int:
    return x * x

Generic Functions

Type parameters are declared in square brackets after the function name:

func identity[T](x: T) -> T:
    return x

Accessing Global Variables

count = 0

func increment() -> void:
    global count
    count += 1

Structs

struct Point:
    x: int
    y: int

func main():
    p = Point(3, 4)
    print(p.x, p.y)

Struct fields are accessed with .. Construction uses the struct name as a function call with positional arguments.

Special Keywords

const

Declares a compile-time constant (constexpr in C++):

const MOD = 1000000007
const PI  = 3.14159

static

Declares a local static — initialized once, persists across calls:

func counter() -> int:
    static n = 0
    n += 1
    return n

defer

Schedules an expression to run at the end of the enclosing function scope (RAII):

func work() -> void:
    defer print("done")
    print("working...")

global

Marks a name as referring to the module-level global:

total = 0

func add(x: int) -> void:
    global total
    total += x

Differences from Python

Deific is intentionally not Python. Three deliberate breaks:

FeaturePythonDeific
Integer typeArbitrary precision64-bit long long
AssignmentReference semantics (GC)Value-copy semantics (no GC)
StringsUnicode codepointsByte strings (std::string)

When Python ergonomics and C++ speed conflict, speed wins.

Examples

Hello World

func main():
    print("Hello World!")

Read and Sort

func main():
    n = read_int()
    nums = read_ints(n)
    nums.sort()
    print(*nums)

Fibonacci

func fib(n: int) -> int:
    if n <= 1:
        return n
    return fib(n - 1) + fib(n - 2)

func main():
    print(fib(10))

Struct Usage

struct Point:
    x: int
    y: int

func dist(p: Point) -> float:
    return float(p.x * p.x + p.y * p.y)

func main():
    p = Point(3, 4)
    print(dist(p))

List Comprehension

func main():
    n = read_int()
    nums = read_ints(n)
    evens = [x for x in nums if x % 2 == 0]
    print(sum(evens))

Modular Exponentiation

const MOD = 1000000007

func main():
    base = read_int()
    exp  = read_int()
    print(pow_mod(base, exp, MOD))
